## Featured Project

### [FinTwit Signal Radar](https://github.com/hy835047657/stock-analysis1)

An open-source Node.js agent that tracks selected FinTwit accounts, classifies market views with an LLM, translates key tweets into Chinese, detects consensus/risk signals, and publishes a daily HTML report with GitHub Actions, Vercel, Supabase, and Lark.

What it does:

- Fetches recent tweets from a configurable watchlist
- Uses an LLM to classify stance, topic, conviction, and risk
- Translates important tweets into Chinese
- Detects multi-blogger consensus and ticker-level risk
- Generates Markdown and HTML daily reports
- Pushes reports to Lark or Slack
- Supports cloud deployment with GitHub Actions + Vercel + Supabase
